# BetterDiscord Plugins
[![Discord Support Server](https://img.shields.io/badge/Discord-Support_Server-5865F2?logo=discord)](https://discord.gg/fKdAaFYbD5)

This repository contains my [BetterDiscord](https://betterdiscord.app) plugins.

## List of Plugins

### GifCaptioner [Page](https://betterdiscord.app/plugins/GifCaptioner) [Source](/plugins/GifCaptioner/)

A plugin that allows you to add captions to gifs from your gif picker.

### ImageFolder [Page](https://betterdiscord.app/plugins/ImageFolder) [Source](/plugins/ImageFolder/)

A plugin that adds a way to quickly upload images from a folder, which can contain subfolders.

### ZipPreview [Page](https://betterdiscord.app/plugins/ZipPreview) [Source](/plugins/ZipPreview/)

Lets you see inside zips and preview/download files without ever downloading/extracting the zip.

### UnicodeEmojis [Page](https://betterdiscord.app/plugins/UnicodeEmojis) [Source](/plugins/UnicodeEmojis/)

Replaces discord emojis that you send with their unicode equivalent.

### VideoCompressor [Page](https://betterdiscord.app/plugins/VideoCompressor) [Source](/plugins/VideoCompressor/)

Easily compress videos that would otherwise be too large to send.

### MoreQuickReacts [Page](https://betterdiscord.app/plugins/MoreQuickReacts) [Source](/plugins/MoreQuickReacts/)

Adds more quick reacts to the message hover popover. Also lets you pin reactions in the popover.

### CssSnippetRepo [Page](https://betterdiscord.app/plugins/CssSnippetRepo) [Source](/plugins/CssSnippetRepo/)

Lets you easily manage CSS snippets which tweak how discord looks.

### PasteAndSend [Page](https://betterdiscord.app/plugins/PasteAndSend) [Source](/plugins/PasteAndSend/)

Adds a "Paste and Send" button for easy pasting and sending without using the keyboard.

## Development

### Setup

Make sure you have [Git](https://git-scm.com) and [Bun](https://bun.sh/) installed. Then, clone this repo and run `bun i --frozen-lockfile` inside of it.

### Structure

The each plugin is located inside `plugins/[name]`, with metadata inside the `config.json` file and code inside the `src` directory. Webpack modules are all centrally stored in `shared/modules`, and plugins need to include a list of modules they use in their config's `modules` field for the bundler. The `.plugin.js` files are all automatically generated by the bundler- they should not be edited directly.

### Building

To build a plugin, run `bun run [name]` (eg `bun run GifCaptioner`). The plugin will then be bundled and output in `plugins/[name]/[name].plugin.js`, and will be automatically copied into your BetterDiscord plugin directory if it is detected. Passing `-t` will leave the script running, and pressing enter in the console will cause it to rebuild and recopy the plugin.
