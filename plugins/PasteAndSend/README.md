# PasteAndSend

A remake of the original [PasteAndSend](https://github.com/Pxrson/BetterDiscord/blob/main/Plugins/PasteAndSend.plugin.js) by ak-tr. Adds a button to the chat bar's context menu which allows for pasting and sending a message without touching the keyboard.
