# CssSnippetRepo

This plugin lets you easily browse and toggle CSS snippets which modify how discord looks in small ways. The number of snippets is still being expanded, but the currently available ones can be seen [here](https://thelazysquid.github.io/DiscordCssSnippets/).

![preview](https://raw.githubusercontent.com/TheLazySquid/BetterDiscordPlugins/refs/heads/main/plugins/CssSnippetRepo/images/preview.png)

## Usage

The snippet manager can be opened by either clicking the palette in the top right corner, or by going to CssSnippetRepo's settings.
